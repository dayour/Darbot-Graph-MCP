Examples

Real-world usage scenarios and code samples

User Provisioning

Automate user account creation and configuration.

// Create new user darbot-graph-users-create displayName: "John Doe" userPrincipalName: "john.doe@company.com" mailNickname: "johndoe" // Assign license darbot-graph-users-license-assign userId: "john.doe@company.com" skuId: "O365_BUSINESS_PREMIUM"

Email Automation

Send automated emails with attachments.

// Send email darbot-graph-mail-send subject: "Weekly Report" body: "

Report

This is the weekly report.

" toRecipients: ["team@company.com"] importance: "High"

Calendar Management

Schedule meetings and manage calendars.

// Create meeting darbot-graph-calendar-events-create subject: "Team Standup" start: "2024-01-15T09:00:00" end: "2024-01-15T09:30:00" attendees: ["alice@company.com", "bob@company.com"]

Teams Collaboration

Post messages and manage Teams channels.

// Send Teams message darbot-graph-teams-messages-send teamId: "team-guid" channelId: "channel-guid" content: "Deployment completed successfully!"

Security Monitoring

Monitor security events and risks.

// List security risks darbot-graph-security-risks-list filter: "riskLevel eq 'high'" // Get audit logs darbot-graph-security-audit-list startDate: "2024-01-01" endDate: "2024-01-31"

Reporting

Generate usage and activity reports.

// Get Teams usage darbot-graph-reports-teams period: "D30" // Get Microsoft 365 usage darbot-graph-reports-usage reportType: "UserActivity" period: "D7"
