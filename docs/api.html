Overview

Darbot Graph MCP provides 64+ tools organized into 10 logical categories. All tools follow the naming pattern: darbot-graph-{category}-{action}

💡

Tool Naming Convention

Tools are hierarchically organized for easy discovery and VS Code's tool limit optimization.

User Management (8 tools)

Complete user lifecycle and directory operations.

darbot-graph-users-list

List users in the directory.

Parameters: filter (string, optional): OData filter expression top (number, optional): Number of results to return orderby (string, optional): Order results by property

darbot-graph-users-create

Create a new user in Azure AD.

Parameters: displayName (string, required): User's display name userPrincipalName (string, required): User's UPN mailNickname (string, required): Mail alias accountEnabled (boolean, optional): Enable account (default: true)

darbot-graph-users-get

Get details of a specific user.

darbot-graph-users-update

Update user properties.

darbot-graph-users-delete

Delete a user from the directory.

darbot-graph-users-password-reset

Reset a user's password.

darbot-graph-users-license-assign

Assign licenses to a user.

darbot-graph-users-manager-get

Get user's manager information.
