Frequently Asked Questions

What is Darbot Graph MCP?

Darbot Graph MCP is an extensible MCP (Model Context Protocol) server that bridges AI assistants with Microsoft 365 services through Microsoft Graph API.

Do I need Azure AD credentials?

No, you can use demo mode for testing. For production use with real Microsoft 365 data, Azure AD credentials are required.

Which VS Code version is supported?

Both VS Code stable and VS Code Insiders with MCP support are compatible.

Can I add custom tools?

Yes! The framework is designed for extensibility. See the Extensibility Guide.

Is this production-ready?

This is a research and development prototype demonstrating MCP integration with Microsoft Graph. Use appropriate security measures for production.

Glossary

MCP (Model Context Protocol)

A protocol for connecting AI assistants to external tools and services.

Microsoft Graph

Unified API endpoint for accessing Microsoft 365, Azure AD, and other Microsoft services.

Azure AD (Azure Active Directory)

Microsoft's cloud-based identity and access management service.

Tool

An MCP-compatible function that performs a specific operation (e.g., sending email, listing users).

Demo Mode

Testing mode that provides simulated responses without requiring Azure AD authentication.
