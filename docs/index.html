Quick Start

🚀 One-Click Installation

The fastest way to get started:

Install for VS Code Install for VS Code Insiders

📦 NPX Installation

Install via NPX:

npx -y @darbotlabs/darbot-graph-mcp

🔧 Manual Installation

Clone and build from source:

git clone https://github.com/dayour/darbot-graph-mcp.git cd darbot-graph-mcp dotnet restore dotnet build

📋 Prerequisites

  • ✅ VS Code or VS Code Insiders with MCP support
  • ✅ .NET 8.0 SDK or later
  • ✅ Azure subscription with Azure Active Directory
  • ✅ Node.js 16+ (for NPX installation)
